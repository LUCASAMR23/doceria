# 🍰 Doceria Celis

> Site desenvolvido com base em uma doceria real, como projeto de portfólio pessoal.

---

## 📋 Sobre o Projeto

A **Doceria Celis** é um site para uma doceria real, desenvolvido como projeto de portfólio. O objetivo foi criar uma experiência de compra online completa, com cardápio, carrinho de compras e responsividade para diferentes dispositivos.

---

## ✨ Funcionalidades

- 🛒 Adicionar e remover itens do carrinho
- 💳 Realizar pedidos/compras pelo site
- 📱 Layout responsivo para mobile, tablet e desktop
- 🍫 Cardápio com os produtos da doceria

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura das páginas
- **CSS3** — estilização e responsividade
- **JavaScript** — interatividade e lógica do carrinho
- **JSON** — armazenamento e manipulação dos dados dos produtos
- **PHP** *(em estudo)* — possível integração com back-end para processamento de pedidos

## 🚀 Como Rodar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/doceria-celis.git
```

2. Acesse a pasta do projeto:
```bash
cd doceria-celis
```

3. Abra o arquivo `index.html` no seu navegador — ou use a extensão **Live Server** no VS Code para uma melhor experiência de desenvolvimento.

> ⚠️ Caso o projeto utilize PHP, será necessário um servidor local como [XAMPP](https://www.apachefriends.org/) ou [WAMP](https://www.wampserver.com/).
